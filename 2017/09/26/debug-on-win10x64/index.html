在 Win10 64位上使用 debug.exe

发表于 2017-09-26 更新于 2021-01-31 阅读次数：

下载DOSBox

从32位机上复制 debug.exe 放到E盘

在 DOSBox 中输入mount c e:\

这步让 DOSBox 用盘符 C 代替访问主机的 E盘

切换到 C 盘，运行
